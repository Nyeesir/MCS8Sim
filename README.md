# MCS-8 computer simulator with built-in assembler and debugger

## How to run the project?

Dependencies:
- Rust (https://www.rust-lang.org/tools/install)

To run the project:
1. Clone the repository
2. Run `cargo run`

To run tests:
1. Clone the repository
2. Run `cargo test`
